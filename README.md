# Healthier Expert System

## Project Overview
Healthier Expert System is a Python-based expert system designed to support basic health assessment and lifestyle planning. The system focuses on Body Mass Index (BMI) calculation, daily energy requirement estimation, and personalized recommendations related to nutrition and physical activity.

## Objectives
- Calculate Body Mass Index (BMI) based on user input  
- Estimate daily energy (calorie) requirements  
- Provide personalized nutrition and activity-based calorie recommendations  
- Demonstrate the application of expert system concepts in a health-related domain  

## Technologies Used
- Programming Language: Python

1. Ensure Python is installed on your system.
2. Open a terminal in the project directory.
3. Run the following command:
